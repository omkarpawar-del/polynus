# Polynus

> A calm, mode-aware daily command center for people balancing Professional, Academic, and Personal life.

Polynus turns approved AI-work context into clear priorities, pending tasks, time visibility, and a focused next move—without forcing everything into one overwhelming task list.

## Why Polynus?

AI work is everywhere: drafting at work, learning new ideas, planning personal projects. But the resulting context is scattered across conversations, tabs, and tools.

Polynus gives that context a home:

- **Professional** — work priorities and commitments
- **Academic** — learning, research, and study tasks
- **Personal** — side projects and life admin

## What the Build Week MVP demonstrates

- Secure Google sign-in with Supabase
- User-approved AI-context import flow
- Task extraction and classification across three life modes
- Duplicate-task reduction in the dashboard
- AI time visibility by mode and source
- Daily Command Center with priorities, a focus plan, and recommendations
- Next.js deployment on Vercel
